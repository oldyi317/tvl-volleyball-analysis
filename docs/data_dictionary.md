## 原始資料格式說明

網站上技術統計以 `"成功 - 總數"` 格式呈現（例如 `"11 - 25"`），
清洗時需拆分為 `_成功` 和 `_總數` 兩個整數欄位。

百分比欄位在網站上以字串呈現（例如 `"44.00%"`），
清洗時需轉換為浮點數。
